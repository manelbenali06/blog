Folder Structure
assets/
						├── css/
						|   |-- icon-list.css
						|   |-- style.css
						|   |-- style.css.map
						├── img/
						├── js/
						├── landing/
						├── plugins/
						├── scss/
						`-- switcher/
  • Spruha - Bootstrap Premium HTML Dashboard Template / : Root template folder contain all html, js, css, scss, images and other files.
  • assets/ : Folder contain all the Admin Template assets which has css, js, scss, and images.
    • css/: Folder contain assets which has complete styles.
      • icon-list : icon-list.css has all template icons link are importated.
      • style : Main style sheet for template.
    • img/ : Template images.
    • js/ : Folder contain all the template pages js files.
    • plugins/ : Folder contain all template plugins.
    • landing/ : Folder contain all landing page styles of template.
    • scss/ : Folder contain all pages scss files which includes dark-theme,rtl,menustyles,headerstyles etc.
    • Switcher/ : Folder contain switcher styles scss files.
  • gulpfile.js : Gulp tasks js file.
  • package.gulp.json : Gulp package json file.
